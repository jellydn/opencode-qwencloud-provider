# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the `opencode-qwencloud-provider` project.

## What are ADRs?

ADRs capture important architectural decisions along with their context and consequences. They help:

- Record the reasoning behind decisions
- Track how the architecture evolves
- Onboard new contributors quickly
- Mark settled decisions as final before moving on

## Conventions

- **Location**: `doc/adr/`
- **Numbering**: Sequential, zero-padded (`000-`, `001-`, …)
- **Filename**: `NNN-title-slug.md`
- **Status lifecycle**: Proposed → Accepted → Deprecated / Superseded by `ADR-N`
